# kiss-rules

收录并生成用于 [KISS-Translator](https://github.com/fishjar/kiss-translator) 的订阅规则。

如果需求某个未收录网站的规则，请提交 [issues](https://github.com/fishjar/kiss-webfixer/issues/new/choose)。

## 部署

部署到了 CloudFlare Page
